# Extra_Os1
Данная программа предназначена для парсинга дампа 1-го сектора таблицы

## Функционал

- Анализ структуры MBR (Master Boot Record)
- Визуализация HEX-дампа первого сектора
- Обнаружение и анализ GPT структуры

## Установка

Для использования программы необходимо загрузить файл zadanie1.exe и запустить его. Помимо этого, проект можно клонировать и запустить его через среду разработки pycharm или другую
